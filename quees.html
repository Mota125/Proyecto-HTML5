¿REALMENTE QUE ES?

HTML es un lenguaje empleado para el desarrollo web y es uno de los famosos y usados, ya que no necesitas ser un super experto para aplicarlo en un proyecto web, cabe recalcar que HTML cuenta con distintas versiones, el HTM4 fue muy empleado pero vino a ser sustituirlo en la actualidad por el HTML5 esta nueva version que surgio; en el año 2014 el cual vino a estandarizar el desarrollo web junto a CSS3 y java script. Incluyendo funciones mas optimizadas para su uso, por ejemplo podemos crear la estructura de una pagina web, texto, imagenes, y material de tipo multimedia que estos funcionarian de una manera mas correcta gracias a esta nueva incorporación. del HTML5.

En la actualidad las grandes empresas que dominan el internet como lo son: Google, Facebook, YouTube, Yahoo, Amazon, Twitter, entre otros sitios web comerciales ya emplean el uso del HTML 5, dado a la factibilidad de su uso, pero obviamente no estan sencillo como parece pero de una manera ha venido a facilitar las tareas a las personas que se dedican a la creacion de sitios web.
